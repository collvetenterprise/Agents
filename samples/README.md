## Android Samples

The Android samples demonstrate how to build M365 agents for mobile devices with:
- Native Kotlin development
- Cross-platform React Native
- .NET MAUI support
- GitHub Copilot integration
- Model Context Protocol (MCP) support
- Enterprise scalability features

For setup instructions, see [ANDROID_SETUP.md](../ANDROID_SETUP.md).
